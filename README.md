# Cooking Competition
Cooking Competition is a fun interactive games with different cooking stations to try and complete orders in a time limit.

### Features
- Cool Art
- Different Stations
- Sound Effects

### Controls
- WASD or Arrows keys to move
- Space to jump
- Mouse to click and drag
- E to interact with objects
  
### How To Play
- HTML (playable in browser)
- Open Itch.IO and search up Fishyyyyyyy_life
- Click the game "Cooking Competition"
- Follow to understand instructions as it is crutial to your game play.
